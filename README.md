# Guess_net

## 项目整体描述几功能
1. 采用C/S的网络编程模型。运行在服务器的服务端程序产生随机数字(目标)，接收客户端的竞猜数字，向客户端发送数字提示信息(大于目标或小于目标)，并统计竞猜次数。
2. 可多个客户端与服务端通信进行竞猜，与服务端商议应用层网络协议(如：数据格式及传输等)
